# blog

#### 介绍
基于Docker的LNMP集成shell安装脚本，使用wordpress搭建个人博客网站。

#### 软件架构
软件架构说明
Nginx 1.15.5  
Php 5.6.36  
MySQL 5.7

#### 安装教程

1.  安装Docker服务
sh install_docker.sh  
2.  安装基于Docker的LNMP集成环境  
sh install_blog_docker.sh  
2.  安装wordpress  
tar -zxvf wordpress-4.9.4-zh_CN.tar.gz -C 【php的存放网页的目录】
